# 📦 Assets — Дополнительные ресурсы

Папка для хранения дополнительных файлов проекта.

## Что можно хранить:

### 📄 Документы:
- Прайс-листы (PDF)
- Презентации
- Сертификаты качества

### 🎨 Векторная графика:
- Логотипы (SVG)
- Иконки (SVG)
- Иллюстрации

### 🎵 Медиа:
- Видео (WebM, MP4)
- Анимации (GIF, Lottie JSON)

### 🔤 Шрифты (если локальные):
- `.woff2` — основной формат
- `.woff` — резервный формат

---

## 🗂 Примерная структура:

```
assets/
├── fonts/
│   ├── oswald-bold.woff2
│   └── roboto-regular.woff2
├── icons/
│   ├── logo.svg
│   └── social-icons.svg
├── documents/
│   ├── price-list.pdf
│   └── certificate.pdf
└── videos/
    └── promo.mp4
```
